# AI Video Note Extractor

An AI-powered web application that converts videos and YouTube lectures into structured notes, transcripts, timelines, and downloadable PDF summaries.

## Features

* Upload video files
* Process YouTube URLs
* Speech-to-text using Whisper AI
* Automatic note generation
* Smart timeline extraction
* PDF export
* Interactive dashboard UI

## Technologies Used

* Python
* Flask
* OpenAI Whisper
* NLP (Sumy)
* Bootstrap 5
* HTML/CSS/JavaScript

## Project Workflow

1. Upload Video / YouTube URL
2. Extract Audio
3. Generate Transcript
4. Create Notes & Summary
5. Build Timeline
6. Export PDF
